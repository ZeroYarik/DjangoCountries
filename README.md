Запуск проекта после клонирования.

1. Создаем виртуальное окружение: python3 -m venv <venv_name> где venv_name название виртуального окружения.
2. Активируем виртуальное окружение: source <venv_name>/bin/activate.
3. Устанавливаем зависимости (их пока нет): pip install -r requirements.txt
4. Создаем базу данных применяя миграции: python manage.py migrate
5. Запускаем сервер командой: python3 manage.py runserver
